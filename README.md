What this lab is:
This lab will walk you through how to build and secure a small network through Cisco Packet Tracer. It will be following the simple security plan that is also in this repository for you to view. The lab will go step by step and afterwards you can do whatever you want with it.

Download these files from the repository:
1. English Translation of Lab Instructions.pdf
2. Starting Network.pkt
3. Security Plan.pdf

Do what is in the lab instructions first then answer the questions in the student survey using this link: https://forms.gle/RrUdqkLe5MYgbrq5A.
