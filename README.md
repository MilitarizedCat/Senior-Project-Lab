What this lab is:
This lab will walk you through how to build and secure a small network through Cisco Packet Tracer. It will be
following the simple security plan that is also in this repository for you to view. The lab will go step by step
and afterwards you can do whatever you want with it.

1. Lab Instructions.pdf
2. Starting Network.pkt
